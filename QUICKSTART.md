## First Time Using the App

### Create Your First Game

1. **Tap "New Game"** on the home screen

2. **Choose Game Rules**
   - Start with "Standard Rules" preset
   - Or customize your own

3. **Add Players**
   - Type names and tap "+"
   - Or select from saved players
   - Need 2-9 players

4. **Tap "Start Game"**

### Enter Scores

1. **Tap "Enter Scores"** button

2. **Enter each player's score**
   - Type manually
   - Or use quick buttons (20, 40, 80)

3. **Tap "Submit"**

4. **Repeat for each round**

Players are automatically eliminated when they reach the max score!
